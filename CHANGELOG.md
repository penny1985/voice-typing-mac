# 更新紀錄 Changelog

## 2026-06（最新）

### 修正
- 加詞視窗的輸入框現在支援 Cmd+C / Cmd+V / Cmd+A / Cmd+X（複製、貼上、全選、剪下）
- 大幅降低結尾「鬼打牆」重複幻覺：關閉 `condition_on_previous_text`，並強化中文長串重複片語的自動清理
- 熱鍵改為「單純點一下才觸發」：右 ⌘ / 右 ⌥ 同時按其他鍵（例如 Cmd+A、Cmd+C）不再誤觸
- 修正按 Caps Lock 或切換輸入法時 App 崩潰的問題（改用 macOS 原生 NSEvent 監看熱鍵）
- 首次下載模型時顯示「⬇️ 下載中」狀態，避免被誤會成當機
- 快速加詞視窗改用浮動視窗，背景 App 也能直接彈到最上層

### 新增
- 錄音波形指示器：錄音時螢幕底部浮出即時音波（白底黑波形）
- 自動分段：講話停頓較久自動換段落、短停頓補頓號
- 慣用語與「錯＝對」修正規則，按右 ⌥ 快速加入
- 中文全形標點、台灣繁體自動轉換
- 歷史紀錄只存記憶體、關閉即清除（保護隱私）
- 一鍵安裝 `install.sh`

## 基礎版
- 完全本機的 Whisper 語音轉文字（mlx-whisper，large-v3-turbo）
- 右 ⌘ 點一下開始／再點一下停止，文字自動貼到游標處
- macOS 選單列 App，全程不連網、免費、開源
